# FiveM-Chat-Commands

Instructions - Installation
1. Unzip from "FiveM Chat Commands"
2. Place "Commands" inside you resources folder
3. Add "start Commands" to you server.cfg // Without the ""
4. Restart Server or Start Scripts Via Console
